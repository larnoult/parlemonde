Bulk edit : users meta fields | posts/CPT categories | Gravity Forms fields

== Description ==

Bulk edit:

* users meta fields
* post and CPT categories
* Gravity Forms entries fields
 
Use new sub menu under users (network level if multisite) : Bulk Edit Users.
Use new sub menu under posts : Bulk Edit Posts.
Use new sub menu under Gravity Forms submenu : Bulk Edit Fields.

== Installation ==

### Easy ###

* Search via plugins > add new.
* Find the plugin listed and click activate.
* Use new sub menu under users (network level if multisite) : Bulk Edit Users
