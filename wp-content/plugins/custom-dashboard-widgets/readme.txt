== Description ==
<h3>Custom Dashboard Widgets</h3>
Hide all default dashboard widgets and add a custom one that show a quick access to popular dashboard links and pages to complete manage your site and your dashboard.<br>
<br>
<h3>Features</h3>
<ul>
<li>Easy to use and more beautiful.</li>
<li>Contains all default links that exist in all wordpress dashboards (Pages, Posts, View Site ... etc).</li>
<li>Allows you to add new links easly.</li>
<li>Allows you to choose icons to the links you added.</li>
<li>Use <a href="http://fortawesome.github.io/Font-Awesome/icons/" target="_blank">fortawesome</a> icons.</li>
<li>Allows you to display links based on wordpress admin roles (admin, author ... etc).</li>
<li>Allows you delete link easly.</li>
<li>Multilingual.</li>
</ul>

== Installation ==

This section describes how to install the plugin and get it working.<br>
1. Upload 'custom-dashboard-widgets' folder to the '/wp-content/plugins/' directory<br>
2. Activate the plugin through the 'Plugins' menu in WordPress<br>

== Frequently Asked Questions ==

= Where can I find the setting after activate the plugin?<br>

You can find the setting throw settings menu -> Dashboard Widgets.<br><br>
= Where can I find the icons?<br>

You can find the icons from this link <a href="http://fortawesome.github.io/Font-Awesome/icons/" target="_blank">Font Awesome </a>.<br>
