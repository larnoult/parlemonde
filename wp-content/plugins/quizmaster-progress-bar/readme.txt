== Description ==

Provides a customizable progress bar for QuizMaster quizzes. Can be integrated and customized from the QuizMaster quiz admin page, various options enable global setup, and supports deep customization at the theme layer.

= Features =

* Visual progress bar shows quiz takers an intuitive view of their progress through a quiz
* Supports variations in how progress is measured including progression towards a passing grade
* Multiple progress bar layouts and themes

== Installation ==

1. Upload the "quizmaster-progress-bar.zip" file via the WordPress's plugin uploader (Plugins -> Add New -> Upload)
2. Activate the plugin through the "Plugins" menu in Wordpress.

== Usage ==

1. Edit or add a new QuizMaster Quiz.
2. Visit the Progress Bar tab in the Quiz Editor
3. Activate Progress Bar for your quiz
4. Customize your Progress Bar with the settings under Quiz > Progress Bar

Also see the Global Progress Bar options under the QuizMaster Settings > Progress Bar including the ability to activate progress bar by default on all new quizzes.

== Frequently Asked Questions ==

= How do I do further customization beyond the option configuration in the QuizMaster Admin? =
All templates found under quizmaster-progress-bar/templates/ can be overridden and customized in your theme. Use CSS styles to further customize the look and feel of the progress bar. For developers deeper customization can be achieved via action and filter hooks.
